네이버 검색광고 API 연동에 필요한 엔드포인트 및 인증 방식 안내입니다.

https://api.searchad.naver.com
인증 방식

⚠ 브라우저에서 직접 API 호출 시 CORS 오류가 발생합니다. Node.js 또는 Python 백엔드 프록시 서버를 통해 API를 호출하고, 이 HTML에서 프록시 주소로 요청을 보내는 구조를 권장합니다.

아래 변수에서 프록시 URL을 설정하세요: const PROXY_BASE = 'https://autobidding-neon.vercel.app/api';
// Node.js 프록시 예시 (express) const express = require('express'); const crypto = require('crypto'); const axios = require('axios'); const app = express(); app.use(express.json()); app.use((req, res, next) => { res.header('Access-Control-Allow-Origin', '*'); next(); }); app.all('/api/*', async (req, res) => { const path = req.url.replace('/api', ''); const ts = Date.now().toString(); const sig = crypto.createHmac('sha256', process.env.SECRET_KEY) .update(`${ts}.${req.method}.${path}`).digest('base64'); const r = await axios({ method: req.method, url: `https://api.searchad.naver.com${path}`, headers: { 'X-Timestamp': ts, 'X-API-KEY': process.env.API_KEY, 'X-Customer': process.env.CUSTOMER_ID, 'X-Signature': sig }, data: req.body }); res.json(r.data); }); app.listen(3000);

1
API 키 발급 필수
네이버 검색광고 시스템(searchad.naver.com)에 로그인 후 도구 → API 관리 → API 키 발급으로 이동합니다. API 키, 시크릿 키, 고객 ID를 메모해 두세요.
💡 API 키는 계정당 최대 3개까지 발급 가능하며 발급 즉시 사용 가능합니다.
2
프록시 서버 실행 필수
API 연동 정보 탭의 Node.js 예시 코드로 로컬 프록시 서버를 실행합니다. 이 HTML 파일 상단의 PROXY_BASE 변수를 프록시 URL로 변경하세요.
⚠ 브라우저에서 네이버 API를 직접 호출하면 CORS 오류가 발생합니다. 반드시 프록시 서버가 필요합니다.
3
API 연결 필수
우측 상단 ⚙ 아이콘을 클릭하여 API 설정 패널을 엽니다. 발급받은 API 키, 시크릿 키, 고객 ID를 입력하고 🔗 연결 버튼을 클릭합니다.
4
키워드 불러오기 추천
키워드 관리 탭에서 캠페인과 광고그룹을 선택한 뒤 📥 키워드 불러오기 버튼을 클릭합니다.
5
희망순위 설정 중요
PC 희망순위와 MO 희망순위를 각각 입력하고 💾 저장을 누릅니다. 이후 갱신 시 이탈 여부를 자동 감지합니다.
✅ PC 3위, MO 5위처럼 기기별로 다르게 설정할 수 있습니다.

통검 순위가 실시간이 아닌가요? ▼
네이버 검색광고 API의 pcNxAvgRnk / mblNxAvgRnk는 실시간 현재 순위가 아니라, 당일 00:00부터 직전 완료된 시각까지의 누적 시간대 평균 순위입니다. 1시간 단위로 업데이트되므로 최대 1시간의 지연이 있습니다.
CORS 오류가 발생하는데 어떻게 해결하나요? ▼
브라우저에서 외부 API를 직접 호출하면 CORS 정책으로 오류가 발생합니다. API 연동 정보 탭의 Node.js 예시 코드로 로컬 프록시 서버를 실행하고, HTML 파일 상단의 PROXY_BASE 변수를 http://localhost:3000으로 변경하세요.
희망순위를 저장했는데 알림이 오지 않아요. ▼
다음 사항을 확인해 보세요:

1. 알림 조건에서 임계값이 너무 높게 설정되지 않았는지 확인
2. 최대 알림 횟수에 도달한 경우 — 키워드 관리 탭에서 알림 초기화 버튼 클릭
3. PC/MO 알림 토글이 OFF 상태인지 확인
4. 저장 후 갱신(↻)을 눌러 최신 데이터를 불러왔는지 확인
자동입찰이 실행됐는데 순위가 개선되지 않아요. ▼
입찰가 변경 후 순위 반영까지 1~2시간의 지연이 발생할 수 있습니다. 순위 데이터 자체가 1시간 평균이기 때문입니다. 경쟁사 상황에 따라 추가 조정이 필요할 수 있습니다.
더보기 순위는 무엇인가요? ▼
네이버 통합검색에서 파워링크 광고는 기본 3~5개가 노출되고, 이후 더보기 영역에 추가 광고가 노출됩니다. 더보기 영역의 순위는 통합검색 상단 노출과 별도로 집계됩니다. 시간대 차트에서 통합검색/더보기 스위치로 구분하여 확인할 수 있습니다.
VAT 포함 표시는 어디서 설정하나요? ▼
